Getting Started
===============

This section outlines how to get started with Kynema-SGF, providing detailed compilation instructions on how to use Exawind-Manager,
a Spack-based package management tool customized for the ExaWind software suite,
of which Kynema-SGF is an integral part. The turbulent flow conditions are established
through precursor simulations, and then turbines are placed in the flow.
